# Aiassistant

Aiassistant is a macOS SwiftUI assistant app with a companion Vite/React showcase site.

## Structure

- `Aiassistant/` - main macOS app source
- `AiassistantTests/` and `AiassistantUITests/` - test targets
- `website/` - React/Vite website
- `example/` - local example project

## Website

```sh
cd website
npm install
npm run dev
```
